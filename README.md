# Video Player 

This is an Online Video player Web application where user can do CRUD Operations.

## Functionality 
- Play Video
- Add Video 
- Modify Video
- Delete Video
- Update Video

## Technology Stack 

- NodeJs
- AngularJs
- ExpressJs
- MongoDb
